# Focule Website

Landing page for [focule.com](https://focule.com)

## Deployment

This site is deployed on Vercel. Any push to `main` will automatically deploy.

## Local Preview

Just open `index.html` in a browser.

## Assets Needed

- `app-screenshot.png` - Screenshot of the Focule app (1200x800px recommended)
- `logo.png` - App logo for header
- `favicon.png` - Browser favicon
